Aodix v4 Features Overview

Aodix v4 is a professional audio sequencer supporting ASIO audio and VST2 plug-ins. It has amazing editing and real-time features. You will be able to create music tracks using Aodix sequencer and external virtual plug-ins that comply with the VST standards (VSTi synthesizers and samplers; VST effects such as reverbs, filters and mastering tools).

Aodix user interface is very close to a tracker's interface, but Aodix is not exactly a tracker. Actually, Aodix v4 is the first software to combine the simplicity of a tracker with the enhancements of a professional sequencer, and much more !

Why a tracker-like interface ?
Because the up/down scrolling of such an interface gives room for a complete parameter display. So, in Aodix v4, each event appears as an object, and all parameters are displayed in a very obvious way! So, the tracker-like interface is a very good starting point for all the features implemented in Aodix, and allows ambitious enhancements, even more that what could be imagined with more common interface systems.

Why more than a tracker :
Traditional trackers only provide step operation, text edition, very low real-time support, very basic routing schemes and many limits. Aodix v4 provides a fully graphic interface with object editing features as well as text-inside-object editing (for almost any parameter)!
Also Aodix v4 is designed with a pure sequencing and hosting architecture. This is the way a sequencer is expected to work today. Sound generating is provided by external VST instruments plug-ins (VSTi synthes, samplers), and DSP is provided by external VST effect plug-ins. All these plug-ins are wired in a modular way allowing any imaginable scheme!
Besides Aodix supports pure real-time, semi-realtime, and offline modes giving you more comfort while composing and editing. Serious real-time operation is made possible with the ASIO implementation.
And, neither last nor least, the multi-pattern object-like support : you may program loops or part of your project in other patterns, and then call and scale them from precise points in your main arrange pattern.

Why much more than a common sequencer :
Aodix handles plug-in modular multi-pin routing, real-time operation, real-time offline editing (while transport is playing somewhere else), high-standard precision and object/text manipulation, in a very wise, enhanced and innovative way. Besides, Aodix supports a wide range of VST plug-ins. This is why Aodix cannot be compared to any common sequencer.

Tracker-like pattern-based sequencer,  with advanced keyboard and mouse based user interface

Aodix v4 user interface provides a layout inspired from trackers. The roll scrolls downwards, as in trackers. Editing is done using mouse and midi and/or PC keyboard. Aodix combines object and range editing for better user-friendly operation.

256 patterns with virtually no length limit, 256 independent markers and cue range

There is virtually no limit for pattern length. A whole song can fit into one pattern. Up to 256 markers can be set in a pattern, and a loop range can be defined inside each pattern.

256 tracks

Up to 256 tracks can be used. A track can hold single events as well as superimposed events (like chords)

Timestamped events with quantization setting and accurate event positioning (up to 960 PPQN). Intelligent step and step-free operating mode

Aodix v4 handles several quantization settings and modes from basic step operation to full step-free operation. According to the recording mode, event position and length are detected and written in an "intelligent" and most "natural" way.

VST2.3 full support, supports a very wide range of plug-ins

Aodix v4 complies to VST2.3 standards, and has been designed to support a very wide range of VST plug-ins (including plug-ins that are known to crash many sequencer software)

Up to 256 instances of VST plug-ins

Aodix allows you to load up to 256 plug-ins in a project, as long as your computer has enough memory installed and enough CPU power.

Modular VST wiring, supporting multi-output

Connect your plug-ins altogether using a visual routing scheme, allowing traditional send/insert scheme and any other scheme you can imagine, using all the inputs/outputs provided by your plug-ins.

Several recording modes, including real-time, editing, and editing while playing

Aodix has several editing and recording modes that you can toggle or combine (common, and less common modes).

VST Automation live recording

In addition to midiCC commands, Aodix is capable of recording VST automation events, that you can trigger with your mouse or with a knob/fader on your midi keyboard. Aodix allows you to assign a knob/fader of your midi keyboard to one or more VST parameters, in one ore more instanced plug-ins.

Full midi message acknowledgment and recording, midi message filtering through mask

Aodix is capable of understanding all midi messages, send them to the plug-ins, and record them. Aodix can filter any kind of messages, according to a fully configurable midi message mask.

Pattern calls (non recursive), with parameter scaling

You may play part of a pattern from another one, just as if it were a simple event, transposing, and scaling velocities.

32-bit float sample precision and IEEE (multi-)file output

Aodix v4 uses the 32-bit float IEEE file format for rendering audio. This format is 32-bit floating-point, mono or multi-channel, and supports a wide range of sampling rates.

ASIO input/output with extremely low latencies

Aodix v4 uses ASIO and achieves very low latencies (in our tests on fast computers : 1ms).
Support for soundcards with no ASIO driver can be obtained by installing a third-party ASIO emulator/wrapper (not supplied).
