Aodix v4 Quick Reference List

Audio Master and Edit :
     - F Keys :      F1 : REC    F2 : Aut     F3 : Loop     F4 : Live     F5 : Stop-Wrap
                          F6 : Step   F7 : Over    F8 : Routing/Sequencer view
    - Spacebar : Play / Stop.
     - Ctrl + 1/2 : set quantize to previous/next value
    - Ctrl + 3/4 : set master channel to previous/next value
    - Ctrl + 5/6 : set octave reference to previous/next value
    - Ctrl + Mouse Wheel : change zoom

Sequencer Browsing:
     - arrows : browse sequencer (elementary)
    - Ctrl + arrows up/down = position cursor exactly on previous/next event in track
    - page up, page down : scroll page up and down
    - Ctrl + page up/down : go to position 000:0/last event position in pattern
    - Tab / Shift + Tab : move to next / previous track
    - Ctrl + left/right arrows : browse 8 tracks left/right
    - Home, End : move to Start/End Cues
    - Double-click (not on object's right section) : set cursor at exact note header location
    - Numpad + / -  : pattern navigation next/previous pattern
    - Enter on a patter call event header : go to the called pattern
    - Ctrl + Enter : get back to the initial pattern

Pattern Arrangement:
     - Del : clear event
    - Insert/Backspace : insert/delete event moving following events in track down/up
    - Ctrl + Insert/Backspace : insert/delete line moving following events in pattern down/up
    - Block operations (to select block, point mouse at left half of a track, draw rectangle or use
      Ctrl + B/E)
            - Ctrl + A : Select all events in the pattern as a block
            - Ctrl + C/X/V : copy/cut/paste block
            - Ctrl + F1/F2 : decrease/increase of 1 unit the byte where the cursor is located, for all the block
                                  (transpose, -1/+1 velocity,...)
            - Ctrl + Q : rearrange block by quantizing notes inside it
    - Object operations (point at right half of an object)
            - Click + drag object : move object
            - Shift + click + drag object : move object, locking vertical moves
            - Ctrl + click + drag object : move object, quantize-free
            - Click at object's bottom + drag up/down = resize object (length)
            - Double-click : delete object
    - Ctrl + Home/End : set Start/End Cues
    - Double-click in the left-most timeline column : create/delete marker
    - Ctrl + double-click in the leftmost timeline column : create/delete STOP marker
    - Special Commands :
            - 1 : write a Jump event
            - Enter : write a midi 0xC program change command (param1 = new program in hex)          
    - Parameter editing : every parameter of an event is editable.

Syntaxes:
    - Note Event syntax: NNN II 00 KK VV
            NNN = note name; II = Instance number (hex); KK = key (hex); VV = velocity (hex)
    - Automation Event syntax : Aut II NN NN VV
            II = Instance number (hex); NN NN = VST Controller Number (hex); VV = value (hex)
    - Pattern call syntax : Pat PP MM TT VV
            PP = pattern (hex) MM = marker (hex) TT = transposing (0x80 = no transpose)
            VV = velocity scaling (0x80 = no scaling, 0x40 = 50%... etc).

Changing values in boxes and knobs :
    - Click and Move mouse on value : change/select value
    - Ctrl + Click and Move mouse on value : tweak value (slower and more accurate).

Routing Operations :
    - Double click in Routing view : load instance
    - Click on instance object and press "Del" : unload instance
    - Click on instance output pin and drag to input pin : draw wire
    - Double-click on the arrow in the middle of the wire : delete wire
    - Ctrl + click/double-click on instance output pin/wire arrow : draw/delete all wires of instance
    - Click and drag on the arrow in the liddle of the wire : adjust wire gain
    - Click on Master Input/Output pin : define audio device for the pin
    - Ctrl + click on the slot of a used instance in the list : center routing view on instance

File Operation : Ctrl + S : Save project.
