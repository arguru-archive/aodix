Troubleshooting


I get no audio : please read carefully the getting started section.

I still get no audio at all : make sure that there is no other Aodix running, and no other software using the same ASIO driver.

I changed my audio drivers and/or audio card, since then Aodix crashes at start : locate the configuration file "Aodix.cfg" and delete it. Open Aodix, configure it from scratch.

Aodix ignores my midi keyboard : make sure no other software is using it, and that you don't have two Aodixes running at the same time...

I try to record notes, but only very short events are recorded : if you are using offline modes, depending on what you are doing, check if Step in the Edit section in on. If you are using small Quantization values, check also that Pre-release is not approximately the same value as Quantize. Pre-Release should be quite smaller than Quantization value (in midi points) for regular offline operations.

I use mouse wheel to scroll up/down, and arrows, but it scrolls very slowly : you are probably using free quantization. Set to a value like 16th.

I have other problems : please browse to our website www.aodix.com and consult the Support section.
